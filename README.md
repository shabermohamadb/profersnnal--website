# Al Ameen Rolling Shutter Manufacturing — UI Scaffold

This is a Vite + React + Tailwind CSS starter scaffold for a premium industrial website.

Quick start:

```bash
npm install
npm run dev
```

Notes:
- Place a cinematic rolling shutter video at `public/video.mp4`.
 - Place a cinematic rolling shutter video at `public/video.mp4`.
 - Google Maps: create a `.env` file in project root with your Google Maps JS API key:

 ```
 VITE_GOOGLE_MAPS_API_KEY=YOUR_API_KEY_HERE
 ```

 Then restart the dev server. If no API key is provided the contact page will render an embed iframe fallback.
- Tailwind is configured in `tailwind.config.cjs`.
